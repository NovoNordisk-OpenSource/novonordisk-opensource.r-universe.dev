# novonordisk-opensource.r-universe.dev
R-Universe for NovoNordisk-Opensource packages

Controls the packages, via `packages.json`, that are made available on [R-universe](https://novonordisk-opensource.r-universe.dev).
See https://docs.r-universe.dev on how to update.

Note the generated repo on R-universe is used in [r.workflows](https://github.com/NovoNordisk-OpenSource/r.workflows) to allow the [check_nn_versions.yml](https://github.com/NovoNordisk-OpenSource/r.workflows/blob/main/.github/workflows/check_nn_versions.yaml) to install the latest version of our own packages.
